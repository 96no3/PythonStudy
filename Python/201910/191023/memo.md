# datetime モジュール

- datetime.date.today()メソッド<br>
  今日の日付を得る<br><br>
- strftime.now()メソッド<br>
  現在の時刻を取得して一定の書式で日時を出力<br><br>
  |書式|意味|
  |---|-------|
  |%Y|西暦（4桁）|
  |%m|月（2桁）|
  |%b|月名を短縮形（Jan,Feb,Mar,...Dec）|
  |%d|日にち（2桁）|
  |%H|24時間表記で時（2桁）|
  |%M|分（2桁）|
  |%S|秒（2桁）|
  |%p|AM/PM（2桁）|
  |%I|12時間表記で時（2桁）|
  |%w|曜日を表す数字（0:日曜,1:月曜,2:火曜,...6:土曜）|
  |%a|曜日名を短縮形（Sun,Mon,Tue,...Sat）|
  |%%|文字'%'|
